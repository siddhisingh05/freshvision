About This Project

Project Summary

FreshnessAI is a unified, AI-based food freshness classification system capable of assessing freshness across multiple food categories — fruits, vegetables, meat, and dairy — using computer vision and deep learning.

Built as a Machine Learning Mini Project (3 Academic Credits) using Flask, TensorFlow / MobileNetV2, SQLite, and deployed on AWS EC2.

Tech Stack
BackendPython 3.10, Flask 3.0
FrontendJinja2 Templates, Bootstrap 5
AI / MLTensorFlow 2.13, MobileNetV2
DatabaseSQLite (via Flask built-in)
CloudAWS EC2, S3 (model storage)
CI/CDGitHub Actions + Docker
